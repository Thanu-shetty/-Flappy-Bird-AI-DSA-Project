# 🐦 Flappy Bird AI + DSA Project

An advanced Flappy Bird game developed using Python and Pygame, integrated with Artificial Intelligence (AI) and Data Structures & Algorithms (DSA) concepts.

This project combines game development with AI automation and DSA implementations for learning and experimentation.


# 🚀 Features

* ✅ Classic Flappy Bird Gameplay
* ✅ AI-Based Bird Training using NEAT Algorithm
* ✅ Data Structures & Algorithms Integration
* ✅ Sorting Algorithms Visualization
* ✅ Binary Tree Implementation
* ✅ Score Management System
* ✅ Interactive Game UI
* ✅ Collision Detection
* ✅ Dynamic Pipe Generation
* ✅ Real-Time Game Physics


# 🛠️ Technologies Used

* Python
* Pygame
* NEAT-Python
* Object-Oriented Programming (OOP)
* Data Structures & Algorithms

# ⚙️ Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/flappy-bird-ai-dsa.git
cd flappy-bird-ai-dsa
```

## 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

## 3️⃣ Run the Game

```bash
python main.py
```


# 🧠 AI Integration

This project uses the **NEAT (NeuroEvolution of Augmenting Topologies)** algorithm to train AI birds automatically.

The AI learns:

* Pipe avoidance
* Timing control
* Jump prediction
* Survival optimization
